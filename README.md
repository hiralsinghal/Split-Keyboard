# Split-Keyboard
A split-keyboard made for ergnomic comfort. It has 42 keys, each side having 21. XIAO-nRF5280 is the MCU powering the project because of its wireless support.

## Features

- Wireless
- XIAO-nRF52840 is the MCU.
- 42 keys
